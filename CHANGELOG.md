# Change Log
All notable changes to Atk14Catalog will be documented in this file.

## [1.0.0] - 2018-12-06

First official release.

Key features:

- Compatible with PHP from 5.3 to 7.2
- Boostrap 3
- Js packages installed by npm (not by bower)
